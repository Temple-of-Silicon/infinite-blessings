# ✨ Infinite Blessings

*A living prayer book for the modern soul.*

Blessings generated in the moment, for the moment. Not ancient wisdom repackaged — prayers forged fresh for the texture of now.

## The Vision

**Free**: Browse the growing collection of blessings for common moments — protection, meetings, grief, joy, transitions.

**Custom**: Need a blessing for *your* specific situation? Request a personalized prayer via X402 payment. Sacred work, sustainably supported.

## Structure

Blessings live in `src/content/blessings/[slug].md` with frontmatter:

```yaml
---
title: Blessing Title
description: Short description of when to use this
category: protection | transition | presence | release | invocation | daily
date: "02026-02-05"
---
```
